## Operating rule

Agents must protect this DTC chain:

```text
Assessment → Diagnosis → Today action → Daily rhythm
```

No agent may expand DTC into a generic CRM, accounting app, debt collection tool, or autonomous customer messaging bot.

## Build rule after this cycle

Every automation cycle must pass:

```text
1. Product Compass check
2. Workflow map
3. Architecture contract
4. Security/RLS review
5. Database review
6. UX/UI plan when user-facing
7. Implementation
8. Test automation/evidence
9. Reality Checker verdict
```

## Immediate result

DTC now has a governance layer for future automation work.

This lets us build:

```text
scheduled owner daily brief
Gmail read-only intelligence adapter
Google Sheets read-only adapter
approved-send queue
Playwright CI
source sync monitoring
```

without guessing, drifting, or bypassing safety.
